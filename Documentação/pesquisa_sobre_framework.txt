Relatório frameworks com CSS

Pesquisei sobre frameworks CSS para ver qual se encaixa melhor no nosso projeto de moda.
O objetivo era encontrar uma ferramenta que ajudasse a criar um site responsivo,
bonito e fácil de usar em diferentes dispositivos.

Os principais que encontrei foram Bootstrap, Tailwind CSS e Materialize CSS.
O Bootstrap é bem popular e tem muita documentação,
mas costuma deixar os sites com cara “padrão”.
O Tailwind CSS é mais flexível, pois usa classes utilitárias que permitem personalizar
bastante o design. Já o Materialize CSS segue o estilo do Google tem componentes prontos,
mas não é tão atualizado e a comunidade é menor.

Com isso, concluí que o Tailwind CSS é a melhor escolha,
pois equilibra modernidade e flexibilidade
O Bootstrap poderia ser usado se quiséssemos algo rápido,
mas não teria o mesmo destaque visual
